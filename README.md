🔗 Compact URL
Compact URL is a full-stack URL shortening application built using the MERN stack. It allows users to generate short URLs and track real-time analytics such as visit counts. The application follows a client–server architecture and is seamlessly deployed on Vercel.

🚀 Features
🔗 Shorten long URLs into compact, shareable links
📊 Real-time analytics to track visit counts for each shortened URL
⚡ Fast and responsive UI
🧩 RESTful API with a scalable backend
🗄️ Persistent data storage using MongoDB

🛠️ Tech Stack
-Frontend
-React.js
-HTML5, CSS3, JavaScript
-Axios

Backend
-Node.js
-Express.js
-MongoDB (Mongoose)

Deployment
-Vercel

🏗️ Architecture
-Client–Server Architecture
-Frontend communicates with backend via REST APIs

Backend handles URL generation, redirection, and analytics

MongoDB stores URLs and visit statistics
